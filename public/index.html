Overview

What Is the CTC Strategy?

The Cross Trend Color (CTC) Strategy is a sophisticated trend-following system built on the Trend Magic framework — combining CCI momentum with ATR volatility — designed for the forex market, primarily EURUSD, using multiple layers of confirmation for high-probability trade signals during peak liquidity sessions.

  • Trend Magic Core — Dynamic trend line from CCI and ATR, color-coded blue (bullish) or red (bearish).
  • Session-Focused — Targets London (03:00–05:00 EST / 13:30–15:30 IST) and New York (08:00–10:00 EST / 18:30–20:30 IST) liquidity windows.
  • HTF Line Conformation — Price must touch the higher timeframe Trend Magic line within the last 3 candles before entry.
  • Smart Entry Levels — Configurable entry at 50% of candle range with auto-validation within 1–2 bars.

Required

Trend Line Crossover (Body)

The entry candle body must cross the Trend Magic line. A wick touching the line does NOT count — the open-to-close body must fully breach it.

Chart Timeframe: Requires 5-minute (M5) chart. All crossover signals are validated on the M5 timeframe.

Trend Color Validation

The Trend Magic line must be blue for buy entries and red for sell entries. This confirms the CCI momentum aligns with the trade direction.

BUY Trend = Blue CCI ≥ 0 → Bullish
SELL Trend = Red CCI < 0 → Bearish
